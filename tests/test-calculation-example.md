# Exemplo de Teste - Cálculo com Valores por Milhão de Tokens

## Valores de Entrada ($/1M tokens)

- **Token de Entrada:** $3.00 por 1M tokens
- **Token de Saída:** $15.00 por 1M tokens
- **Token de Cache:** $1.50 por 1M tokens

## Textos de Exemplo

### Texto de Entrada:

```
Explique como funciona a inteligência artificial e suas principais aplicações no mundo moderno.
```

### Texto de Saída:

```
A inteligência artificial (IA) é uma tecnologia que permite que máquinas simulem a inteligência humana através de algoritmos e processamento de dados. Suas principais aplicações incluem: reconhecimento de voz, processamento de linguagem natural, visão computacional, sistemas de recomendação, carros autônomos, diagnósticos médicos, assistentes virtuais, análise preditiva, automação industrial e jogos estratégicos. A IA funciona através de redes neurais artificiais que aprendem padrões nos dados e fazem previsões ou tomam decisões baseadas nesses padrões.
```

## Estimativa de Tokens

- **Tokens de Entrada:** ~18 tokens (15 palavras ÷ 0.75)
- **Tokens de Saída:** ~95 tokens (71 palavras ÷ 0.75)
- **Tokens de Cache:** ~2 tokens (10% dos tokens de entrada)

## Cálculo de Custos (Quantidade: 1)

### Conversão para Preço por Token Individual:

- Entrada: $3.00 ÷ 1,000,000 = $0.000003 por token
- Saída: $15.00 ÷ 1,000,000 = $0.000015 por token
- Cache: $1.50 ÷ 1,000,000 = $0.0000015 por token

### Custos Individuais:

- **Custo Entrada:** 18 × $0.000003 = $0.000054
- **Custo Saída:** 95 × $0.000015 = $0.001425
- **Custo Cache:** 2 × $0.0000015 = $0.000003

### Custo Total:

**$0.001482** por operação
